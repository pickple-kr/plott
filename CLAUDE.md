# PLOTT 프로젝트

## 이 프로젝트가 뭔지
식물 마켓플레이스 겸 커뮤니티 웹사이트.
사람들이 사업자 인증하고 가입해서 자기 식물을 등록하고,
구매자는 둘러보다가 "구매하러 가기" 버튼으로 판매자의 외부 판매 링크로 이동한다.
플랫폼은 직접 결제·정산을 하지 않는다 (수수료 없음).
대신 사람을 모아서 PLOTT 자체 상품 노출과 광고로 수익을 낸다.
커뮤니티 게시판도 있어서 식물 자랑·고민 상담·정보 공유를 할 수 있다.

## 나에 대해
- 나는 코딩을 전혀 모른다. 항상 한 단계씩, 쉬운 말로 설명해줘.
- 실행할 명령어는 내가 직접 복사해서 칠 수 있게 하나씩 알려줘.
- 한 번에 여러 기능을 만들지 말고, 한 조각씩 만들고 확인하면서 가자.
- 에러가 나면 무슨 뜻인지 쉽게 풀어서 설명해줘.

## 기술 스택
- Next.js (App Router)
- Tailwind CSS
- Supabase (DB + 로그인 + 이미지 저장)
- Vercel 배포

## 만들 순서 (로드맵)
1. 프로젝트 기본 뼈대  ← 지금 여기
2. Supabase 연결 (로그인 + DB)
3. 식물 등록·둘러보기 (마켓)
4. 커뮤니티 게시판 (카테고리: 자랑/고민/팁/자유)
5. 관리자 (사업자 인증 승인)
6. 디자인 입히기
7. Vercel 배포
